# Paulet Labs

Personal project index for **Charlie Normand**, based in South East London.

This static site serves as the homepage for pauletlabs.co.uk, showcasing both legacy projects and more recent work created with [Claude Code](https://claude.ai/code).

## Features

- Bold, modern design with strong typography
- Historical Gen-AI created 1910 Paulet Road photograph as backdrop
- SPA-like smooth navigation between sections
- Categorized project index (Legacy / Claude Code)

## Development

Open `index.html` in a browser, or serve locally:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
